# Strategic Product Placement Analysis: Unveiling Sales Impact with Tableau Visualization
 
**Project Name:** Strategic Product Placement Analysis: Unveiling Sales Impact with Tableau Visualization  

## Project Overview
In the Fast-Moving Consumer Goods (FMCG) sector, retail managers struggle to quantify the combined impact of product placement, foot traffic, and promotional activities on sales. This project delivers a Data-Driven Executive Analytics Portal using Tableau and Python (Flask). By processing historical retail data, this system uncovers hidden correlations (such as the "Promotion Paradox") and empowers managers to optimize shelf space confidently.

## Technology Stack
* **Data Analytics & Visualization:** Tableau Desktop / Tableau Public API
* **Backend Web Server:** Python (Flask, Pandas, Werkzeug)
* **Frontend User Interface:** HTML5, CSS3, Bootstrap 5 
* **Data Storage:** Flat-file CSV (Product Positioning.csv)

## How to Run the Application
1. Download the `Project Development Phase` folder.
2. Install the required Python packages: `pip install flask pandas`
3. Run the Flask server: `python app.py`
4. Access the web portal locally at `http://127.0.0.1:5000`
